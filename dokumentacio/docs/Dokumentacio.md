## Dokumentáció

## Összegzés

A projekt jelenleg két külön egységből épül fel: egy frontend és egy backend alkalmazásból. A frontend React Native technológiával készült mobilalkalmazás, amely egy listanézetben jelenít meg 20 darab előre definiált (mockolt) adatot. A backend oldalon egy Express.js alapú rendszer működik, amely egy REST API segítségével kezeli a cikkekkel kapcsolatos műveleteket.

### Projekt feladata

A fejlesztés következő lépéseként a két alkalmazás összekapcsolása a cél, hogy a backend által szolgáltatott adatok megjelenjenek a mobilalkalmazás felületén. A tervezett megoldás egy végtelen görgetéses (infinite scroll) alkalmazás, amely működésében a közösségi média platformokhoz – például a Twitter (X) – hasonló élményt nyújt. A projekt gyakorlati példán keresztül mutatja be a frontend és a backend együttműködését egy valós alkalmazási környezetben.
